The software required to develop, integrate, and operate a spacecraft is substantial. OpenSatKit integrates NASA's Core Flight System (cFS) with Ball Aerospace's COSMOS ground system so that you can start your mission faster.

Install (Ubuntu only)

$ bash <(\wget -qO- https://raw.githubusercontent.com/OpenSatKit/OpenSatKit/vendor/install.sh)_
